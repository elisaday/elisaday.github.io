Ubuntu-16.04 安装Tensorflow-GPU

发表于 2019-04-03

安装nVidia显卡驱动

在官网下载驱动，直接安装就可以。需要注意的是，在安装的时候，如果提示gcc版本不一致，那么最好安装一致版本号的。并且在后面需要添加参数–no-opengl-files

Installing CUDA 7.5 with Ubuntu 16.04 or Ubuntu 14.04

安装CUDA

在官网下载工具包，1个主包，4个补丁包，依次安装。注意版本号为9.0。

安装cuDNN

在官网下载以后，解压缩，再把里面的cuda目录拷贝到上面CUDA的安装位置，默认为：/usr/local/cuda/。注意，下载的版本号需要和上面CUDA版本号保持对应。

配置路径

安装Tensorflow-GPU

注意：1.12版本的Tensorflow-GPU才是和上面9.0版本的CUDA对应的。如果安装1.13版本的Tensorflow-GPU，那么会调用10.0版本的CUDA。
