Installing CUDA 7.5 with Ubuntu 16.04 or Ubuntu 14.04

安装CUDA

在官网下载工具包，1个主包，4个补丁包，依次安装。注意版本号为9.0。

安装cuDNN

在官网下载以后，解压缩，再把里面的cuda目录拷贝到上面CUDA的安装位置，默认为：/usr/local/cuda/。注意，下载的版本号需要和上面CUDA版本号保持对应。

安装Tensorflow-GPU

图片处理公式

发表于 2018-10-19

调整亮度和对比度

$g(x)=\alpha \times f(x) + \beta$

  • $g(x)$ 输出像素
  • $f(x)$ 输入像素
  • $\alpha$ 对比度
  • $\beta$ 亮度
  • x 像素位置

RGB转灰度

  • $Gray = (R + G + B) \div 3$
  • $Gray = R \times 0.3 + G \times 0.59 + B \times 0.11$
  • $Gray = R \times 0.2126 + G \times 0.7152 + B \times 0.0722$
  • $Gray = R \times 0.299 + G \times 0.587 + B \times 0.114$
  • $Gray = ( \max (R, G, B) + \min (R, G, B) ) \div 2$
  • $Gray = \max (Red, Green, Blue)$
  • $Gray = \min (Red, Green, Blue)$
